# LSPP 306 (Tiga Puluh Juni) - Portal Lembaga Sertifikasi Profesi

LSPP 306 adalah platform digital resmi yang dirancang untuk mendukung komitmen lembaga dalam meningkatkan standar kompetensi dan profesionalisme tenaga kerja di Indonesia. Sebagai mitra strategis bagi individu dan organisasi, platform ini memfasilitasi kebutuhan sertifikasi sesuai standar nasional yang berlaku.

## 📋 Tentang LSPP 306

Dengan pengalaman lebih dari satu dekade, LSPP 306 telah melayani ribuan peserta sertifikasi dari berbagai sektor industri. Beroperasi di bawah lisensi resmi **Badan Nasional Sertifikasi Profesi (BNSP)**, menjamin setiap sertifikat memiliki nilai pengakuan tinggi di pasar kerja nasional maupun internasional.

Sistem ini dibangun untuk memberikan layanan uji kompetensi yang:

- **Profesional**: Proses yang terstandarisasi.
- **Objektif**: Penilaian yang adil dan transparan.
- **Dapat Dipercaya**: Menjaga kredibilitas bagi seluruh pemangku kepentingan.

## 🚀 Fitur Utama

- **Sertifikasi Pariwisata**: Program sertifikasi resmi untuk meningkatkan kompetensi industri pariwisata.
- **Landing Page Dinamis**: Hero section dengan slider interaktif dan navigasi modern.
- **Informasi Skema & Asesor**: Detail mengenai skema sertifikasi, daftar asesor, dan Tempat Uji Kompetensi (TUK).
- **Berita Terbaru**: Update terkini mengenai kegiatan dan pengumuman lembaga dengan fitur _infinite auto-scroll_.
- **Klien & Mitra**: Menampilkan ekosistem mitra yang telah bekerja sama dengan LSPP 306.

## 🛠️ Stack Teknologi

Proyek ini dikembangkan menggunakan teknologi modern untuk performa dan SEO yang optimal:

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Shadcn/UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: CSS Marquee & Framer Motion (Optional)

## 📦 Instalasi

Ikuti langkah-langkah berikut untuk menjalankan proyek di mesin lokal Anda:

1.  **Clone repositori**

    ```bash
    git clone [https://github.com/username/lspp306.git](https://github.com/username/lspp306.git)
    cd lspp306
    ```

2.  **Instal dependensi**

    ```bash
    npm install
    # atau
    yarn install
    ```

3.  **Jalankan server pengembangan**

    ```bash
    npm run dev
    ```

4.  **Buka di browser**
    Akses [http://localhost:3000](http://localhost:3000) untuk melihat hasilnya.
